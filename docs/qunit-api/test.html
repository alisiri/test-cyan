test( title [, expected ], test )

Description: Add a test to run.

  • test( title [, expected ], test )

    • title
      Type: String
      Title of unit being tested
    • expected
      Type: Number
      Number of assertions in this test
    • test
      Type: Function( QUnit.assert:assert )
      Function to close over assertions

When testing the most common, synchronous code, use test()

The assert argument to the callback contains all of QUnit's assertion methods. If you are avoiding using any of QUnit's globals, you can use the assert argument instead.

Examples:

Example: Simple, synchronous code test.

test( "hello test", function() {
  ok( 1 == "1", "Passed!" );
});

  function square( x ) {
    return x * x;
  }
 
  var result = square( 2 );
 
  assert.equal( result, 4, "square(2) equals 4" );
});
