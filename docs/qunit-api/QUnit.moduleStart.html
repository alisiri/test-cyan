QUnit.moduleStart( callback )

Description: Register a callback to fire whenever a module begins.

  • QUnit.moduleStart( callback )

    • callback
      Type: Function( Object details )
      Callback to execute. Provides a single argument with the following property:
      • name
        Type: String
        Name of the next module to run

Example:

Register a callback that logs the module name

1
2
3
 
QUnit.moduleStart(function( details ) {
  console.log( "Now running: ", details.name );
});
