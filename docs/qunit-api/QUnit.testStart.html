QUnit.testStart( callback )

Description: Register a callback to fire whenever a test begins.

  • QUnit.testStart( callback )

    • callback
      Type: Function( Object details )
      Callback to execute. Provides a single argument with the following properties:
      • name
        Type: String
        Name of the next test to run
      • module
        Type: String
        Name of the current module

Example:

Register a callback that logs the module and name

1
2
3
 
QUnit.testStart(function( details ) {
  console.log( "Now running: ", details.module, details.name );
});
