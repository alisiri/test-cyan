start( [decrement ] )

Description: Start running tests again after the testrunner was stopped. See stop().

  • start( [decrement ] )

    • decrement
      Type: Integer
      Optional argument to merge multiple start() calls into one. Use with multiple corrsponding stop() calls.

When your async test has multiple exit points, call start() for the corresponding number of stop() increments.

Example:

A test that waits 150ms for an async operation to finish, then asserts the result.
